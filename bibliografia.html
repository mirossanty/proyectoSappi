1¿Qué software se emplearon para el desarrollo del proyecto?

Arduino: Desarrollo de código para la programación del bastón y los lentes.
TinkerCad: Pagina para la realización de las simulaciones para el código del bastón y los lentes.

2¿Cuánto tiempo empleo para el desarrollo del proyecto?

El sistema en general fue desarrollado de forma intermitente, sin embargo, de ser realizado de forma consecutiva tomaría de 2 a 3 meses.

3¿A qué compañía pertenecen la mayor parte componentes que se emplearon en el proyecto?

Las placas empleadas para el desarrollo del bastón y los lentes son Arduino, ya que permite una forma de programación sencilla por lo que la lógica respecto al código se vuelve una herramienta indispensable para que las indicaciones sean clara y sencilla.
